# 🎉 Happy Birthday Surprise Website 💖

A beautiful, interactive, and heartfelt birthday website built to celebrate someone special. This project combines emotional messages, fun interactions, animations, music, and memorable surprises into a single responsive website.

---

## ✨ Features

* 🎂 Beautiful landing page
* 💌 Heartfelt birthday wishes
* ❤️ Emotional love letter
* 😂 Funny and memorable moments
* 🖼️ Photo gallery
* ⏳ Countdown timer
* 🎆 Confetti and fireworks animations
* 💖 Floating hearts and visual effects
* 🎵 Background music
* 🎁 Surprise section
* 💍 Proposal section
* 📱 Fully responsive design
* ⚡ Smooth animations and transitions

---

## 🛠️ Built With

* HTML5
* CSS3
* Vanilla JavaScript

No frameworks or libraries were used.

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/birthday-website.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

That's it! No installation or additional setup is required.
